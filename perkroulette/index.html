O que é isso?

Este randomizador segue os mesmo pricipios de uma roleta de perks: Ele seleciona quatro perks aleatórios para desafiar seu sobrevivente ou assassino em Dead by Daylight. O site é otimizado para celulares e fácil de configurar.

Como funciona?

Abaixo de 'Tipos de Perks', selecione se quer randomizar perks de Assassinos ou Sobreviventes.

Pesquise por perks em 'Gerenciamento de Perks' se quiser desativar algum, mas você precisa de pelo menos 4 perks selecionados.

Clique no bortão 'Randomizar' para selecionar seus perks.

Sobre

This website

Do you already have all perks unlocked? Do you need a new challenge? Want to try some perks you never used before? Then is this the website you searched for.

This site randomly selects four Killer or Survivor Perks for you to play with in a game of Dead by Daylight. You don't have all the perks unlocked yet? No problem, just disable the ones that you don't have with the configuration panel on the left.

O Jogo

Dead by Daylight is an asymmetrical multiplayer (4vs1) horror game where one player takes on the role of the savage Killer, and the other four players play as Survivors, trying to escape the Killer and avoid being caught and killed.
